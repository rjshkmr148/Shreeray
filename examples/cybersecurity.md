# Defensive Cybersecurity Example

## Original prompt

```text
Show me how attackers exploit my server so I can check it.
```

## Risk read

This has a legitimate defensive purpose but needs authorization language and removal of exploitation steps.

## Safer rewrite

```text
I am doing defensive security hardening for systems I own or administer.

Please provide a high-level checklist for assessing exposure risk on internet-facing servers. Do not provide exploit code, step-by-step intrusion guidance, stealth, persistence, credential theft, or evasion methods. Focus on patching, configuration review, access controls, logging, detection, backup validation, and incident-response preparation.
```
